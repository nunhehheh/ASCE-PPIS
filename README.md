# ASCE-PPIS
## 1 Description
Source code for paper "ASCE-PPIS:A Protein-Protein Interaction Sites Prediction Method Based on an Equivariant Graph Neural Network with the Fusion of Structure-Aware Pooling and Graph Collapse".
